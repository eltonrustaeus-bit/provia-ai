Matematik C · 2026-05-17
0/2 p
Visa med pq-formeln att x² − 4x + 3 = 0 har lösningarna x = 1 och x = 3.
Ditt svar
x = 1
Modellsvar
p = −4, q = 3.  x = 4/2 ± √((4/2)² − 3) = 2 ± 1.  Alltså x = 1 och x = 3.
