Vad vi samlar in

ProviaAi samlar in minimal data för att kunna erbjuda tjänsten. Vi använder inga spårningscookies, inga annonsnätverk och inga tredjepartsanalysverktyg.

  • Kontoinformation: e-postadress och lösenord (hashad) vid registrering. Lagras i Supabase (EU-region).
  • Studieprogress: dina provresultat, felbanken och kategoriframsteg. Lagras i Supabase kopplat till ditt konto.
  • Lokalt på din enhet: inloggningssession, temainställning, cached AI-förklaringar. Se tabellen nedan.

Cookies & lokal lagring

Vi använder localStorage (webbläsarlagring, inte traditionella cookies) för att spara din session och inställningar. Inga data delas med tredje part för marknadsföring.

Tredjepartstjänster

  • Supabase (databas & auth) — EU-region (Frankfurt). Supabase integritetspolicy
  • OpenAI (AI-förklaringar) — dina svar skickas till OpenAI för att generera förklaringar. OpenAI integritetspolicy
  • Stripe (betalning) — om du uppgraderar. Kortuppgifter hanteras enbart av Stripe. Stripe integritetspolicy
  • Vercel (hosting) — serverloggar inkl. IP-adress (ingen spårning). Vercel integritetspolicy
  • Google Fonts — typsnitt laddas från Google. Google integritetspolicy
